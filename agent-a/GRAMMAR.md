# Grammar Rules

## Numbers
- N× = quantity, (N) = step/size label

## Shapes
- △ □ ○ · ★ ◉

## Arrangement / Spatial
- ←→ horizontal, ↕ vertical, X[Y] = Y inside X, X⊂Y = X contained in Y

## Colors
- 🔴 🔵 🟢

## Actions
- +N→/↓ = move, ↻ = rotate, ×N = size multiply

## Sequences
- ... = continues, ?= = prediction, [XY] = repeat unit

## Conditionals
- X ⟹ Y = if X then Y

## Emotions
- ⚠☠ ←←← = danger/avoid, ❓🔍 →→→ = curiosity/explore

## Negation
- 🚫X ¬X = NOT X, ✗ rejected, ∉ not inside

## Temporal
- ⏱ = time, ∅ = nothing, (1)(2)(3) = steps

## Interrogative
- ❓ = question, ♥ = favorite, ╱ = side, ?×╱ = how many sides

## Back-reference (new for level 14)
- @NNN-x = reference to message NNN-x
- @006-a + @006-b = combine elements from those messages

## Confirmation
- ✓ = understanding
