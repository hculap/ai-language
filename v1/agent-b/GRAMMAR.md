# Grammar Rules

## Shape Vocabulary
- △ = triangle, □ = square, ○ = circle, · = dot/point, ★ = star

## Color
- 🔴 = red, 🔵 = blue, 🟢 = green

## Quantity & Composition
- `N×[shape]` = N instances. Repeat symbol for count.

## Spatial/Direction
- ←→ = horizontal, ↕ = vertical, ↑ = above, ↓ = below
- ⊃ or `[...]` = contains/inside. ──── = line/divider.

## Actions/Transforms
- → = "then". ↻ = rotate, ×2 = scale, +N→/+N↓ = translate.

## Patterns & Sequences
- `...` = continues, `?=` = prediction, `+N` = increment

## Conditionals/Logic
- ⟹ = conditional "then". ≤ / > = comparison.

## Negation
- ¬ = NOT, ∉ = not contained, 🚫 = visual negation

## Temporal
- ⏱ = time, ∅ = nothing, (N) = step number

## Interrogatives
- ❓: = question prefix, ♥ = placeholder, ? = unknown

## Back-Reference (Level 14)
- 📎NNN-x = reference to message NNN-x
- ◇ = modified version of referenced message
- Show original → show changes → show result

## Abstract Concepts
- ⚠ ⟹ ←←← (danger), ❓ ⟹ →→→ (curiosity)

## Feedback
- ✓ = confirmation
