# Grammar Rules

## Numbers
- N× prefix = quantity multiplier

## Shapes
- △ = triangle, □ = square, ○ = circle

## Arrangement
- ←→ = horizontal, ↕ = vertical
- ↑/↓ relative to ──── = above/below line

## Spatial
- X[Y] = Y inside X, X > Y = X larger

## Colors (new for level 06)
- 🔴 = red, 🔵 = blue, 🟢 = green
- Color prefix before shape: 🔴○ = red circle
- ♥ = red, ♦ = blue (alternate)

## Composition
- N×(color)(shape) + M×(color)(shape)
- Example: 2×🔴○ + 1×🔵□

## Confirmation
- ✓ = understanding/acknowledgment
