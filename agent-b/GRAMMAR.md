# Grammar Rules

## Shape Vocabulary
- △ = triangle, □ = square, ○ = circle, · = dot/point

## Color
- 🔴 = red, 🔵 = blue, 🟢 = green
- Color precedes shape: 🔴○ = red circle, 🟢△ = green triangle

## Quantity & Composition
- `N×[shape]` = N instances of a shape.
- `N×[color][shape] [direction]` = full composition.

## Size
- Repeat shape symbol for relative size: △ < △△ < △△△
- ⬆ = growing/increasing

## Spatial/Direction
- ←→ = horizontal, ↕ = vertical
- ↑ = above, ↓ = below
- ⊃ or `[...]` = contains/inside
- ──── = line/divider

## Feedback
- ✓ = confirmation, → = "maps to"
